# Deep Video Research

## Installation

```bash
pip install -r requirements.txt
pip install gradio
```

Set your OpenAI API key:
```bash
export OPENAI_API_KEY="your-api-key-here"
```

## Usage

### Command Line
```bash
python local_run.py "https://www.youtube.com/watch?v=VIDEO_ID" "your question about the video"
```

### Web UI
```bash
python app.py
```
